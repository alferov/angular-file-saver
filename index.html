Angular File Saver

An AngularJS service that provides cross-browser compatibility of the HTML5 saveAs()

View on GitHub Download .zip Download .tar.gz

Installation

Bower

$ bower install angular-file-saver

Basic usage

  1. Include the fileSaver module to yor project;
  2. Pass SaveAs service as a dependency;
  3. That's it!
