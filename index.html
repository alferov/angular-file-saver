Angular File Saver

An AngularJS service that provides cross-browser compatibility of the HTML5 saveAs()

View on GitHub Download .zip Download .tar.gz

Installation

Using bower

$ bower install angular-file-saver

Using npm

$ npm install angular-file-saver

Dependencies

  • AngularJS
  • FileSaver.js
  • Blob.js

Basic usage

  1. Include the ngFileSaver module to yor project;
  2. Pass FileSaver service as a dependency;
  3. Invoke FileSaver.saveAs() and pass an object with the following set of options:
    • data - data represented as an array or a Blob object;
    • filename
    • options - set of options for the Blob constructor (optional attribute)

Example:

function ExampleCtrl($scope, FileSaver) {
  var vm = this;

  vm.val = {
    text: 'Hey ho lets go!'
  };

  vm.download = function(text) {

    var config = {
      data: [text],
      filename: 'textfile.txt',
      options: {
        type: 'text/plain;charset=utf-8'
      }
    };

    FileSaver.saveAs(config);
  };
}

angular
  .module('fileSaverExample', ['ngFileSaver'])
  .controller('ExampleCtrl', ['$scope', 'FileSaver', ExampleCtrl]);
